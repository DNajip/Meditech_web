Documento de Task List
Proyecto: MediTech – Sistema de Gestión Clínica

Versión: 1.0
Tipo de proyecto: Software Web
Arquitectura: ASP.NET Core MVC + SQL Server

1. Objetivo del documento

Este documento define la lista de tareas necesarias para desarrollar el sistema MediTech.

El objetivo es organizar el desarrollo del software en actividades claras que permitan implementar el sistema de forma progresiva, ordenada y escalable.
